Technical Architecture

系统架构

🖥️ 前端展示层

HTML5 大屏 · ECharts 图表 · 模块 Tab · 图层控制面板

Vite 5ECharts 5.5CSS Variables

🌐 WDP 云渲染层

3D 引擎 · WebRTC 视频流 · 覆盖物 API（POI/热力/路径/区域）

WDP API 2.3WebRTCCloud Render

📡 数据服务层

RESTful API · WebSocket 推送 · 数据库 · 告警引擎

REST APIWebSocketPostgreSQL

数据流向

① 前端发起 API 请求获取业务数据
② WDP 引擎接收指令渲染 3D 场景
③ WebRTC 实时推送视频流至浏览器
④ WebSocket 推送告警事件触发联动
⑤ 覆盖物 API 叠加 POI/热力/路径/区域

核心优势

▸ 云端渲染，终端零门槛（浏览器即用）
▸ 零插件，WebRTC 原生传输
▸ 覆盖物与场景解耦，独立图层控制
▸ 支持多端：大屏 / PC / 平板 / 手机

Value Proposition

项目价值

👁️

全域可视

校园全景一屏掌握，38栋建筑空间化呈现。从宏观俯瞰到微观建筑详情，多层级数据融合立体展示，让管理者"看见"整个校园。

⚡

实时响应

告警秒级推送，场景自动飞行聚焦。从发现到处置全链路可追溯，平均响应时间从分钟级降至秒级，让应急不再被动。

🧠

智能决策

AI 视频感知 + 多维数据分析，自动识别异常模式。16+ 业务图表辅助管理决策，从"经验驱动"升级为"数据驱动"。

🔗

统一平台

安防·运维·检测·概览四合一，打破信息孤岛。云渲染零门槛，浏览器即用，大屏/PC/平板/手机多端适配，无需安装任何插件。
